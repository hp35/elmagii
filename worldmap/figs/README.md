# Figure for the "World Map of Classical Electromagnetics"

This repository contains the MetaPost code for the "World Map of Classical
Electromagnetics", used in the series of lectures on Electromagnetic Theory
II (1TE626) for Engineering Physics (master level) at Uppsala University.

## Compiling the MetaPost figures

Just run `make` and ensure to install any missing components in case there
are any complaints.
