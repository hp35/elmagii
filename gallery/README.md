# Portrait gallery in classical electromagnetism

Electromagnetism as a subject is packed with celebrities from classical physics,
and it is always nice to have a face associated with a name or equation. This
portrait gallery covers the primary discoverers in electrostatics,
magnetostatics, and electrodynamics.
